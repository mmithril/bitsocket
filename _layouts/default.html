Bitsocket is WebSocket server implementation for crypto-currency exchanges. It enables exchanges to implement and run their own streaming API in short period of time.

Bitsocket is written in pure Java. Event driven and non blocking way ensures that it handles a lot of concurrency using a small number of kernel threads. It scales with minimal hardware requirements for the largest deployments.
